# 网页内容变动提醒

## 介绍

这是一款支持当指定的网页或接口响应发生变化时在桌面右下角弹出提醒框的小程序

## 截图

![image](/Screenshots/MainWindow.png)

## 特性

- 使用.Net Framework 4.8 + WPF 开发
- 使用MD风格的界面
- 支持GET和POST
- 支持HTML和JSON
- 支持中文和英文，支持扩展更多语言
- 能够跟随系统亮色/暗色模式进行自动切换
